# 🤖 Aria — Voice & Text AI Chatbot

A Python-based voice and text chatbot built with Speech Recognition, Text-to-Speech, Wikipedia API, and live Weather API.

## ✨ Features
- 🎤 Voice input via microphone
- 🔊 Speaks responses aloud
- 🌤️ Live weather updates
- 📖 Wikipedia-powered answers
- 😄 Jokes & fun facts
- 📁 Conversation logging

## 🛠️ Installation
pip install speechrecognition pyttsx3 pyaudio requests

## ▶️ Run
python chatbot.py

## 💬 Commands
- Say **"joke"** → get a joke
- Say **"fact"** → get a fun fact  
- Say **"weather in Kerala"** → live weather
- Say **"time"** or **"date"** → current time/date
- Type **"voice"** → switch to mic input
